# 📊 Análise Socioeconômica do Mercado de Trabalho com Machine Learning

## 🎯 Objetivo
Analisar dados da RAIS para prever cenários socioeconômicos utilizando modelos de Machine Learning.

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Scikit-learn

## 📈 Modelos aplicados
- Random Forest
- Regressão Linear
- Gradient Boosting
- KNN
- Redes Neurais

## 📊 Resultados
O modelo Random Forest apresentou melhor desempenho com R² próximo de 0.99.

## 📂 Base de dados
RAIS – Ministério do Trabalho

## 🚀 Insights
Foi possível identificar padrões relevantes nas admissões e prever tendências futuras do mercado de trabalho.
